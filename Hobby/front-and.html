Mengapa Saya Tertarik pada Dunia IT ?

Ketertarikan saya pada dunia Teknik informatika itu berawal dari saya menjadi korban adu domba dari si hecker, Tepatnya di pertengahan bulan Mei tahun 2020 saat bulan puasa dan saat itu juga dunia di landa oleh virus covid-19 / corona. Saat itu sekolah-sekolah menerapkan sistem Pembelajaran Jarak Jauh.


Pada pagi hari saat saya selesai absen sekolah lalu ingin mengerjakan tugas yang diberikan oleh guru, lalu masuklah pesan dari murid yang terdapat pada grub sekolah. Isi pesannya itu mengandung kata-kata kasar yang membuat orang tua saya tidak terima, lalu orang tua saya mengirim pesan ke orang tersebut lewat grup sekolah karena pesan yang dikirimkan oleh murid tersebut sudah saya hapus karena menurut saya pesan tersebut tidaklah penting.


Kemudian orang tua dari si murid tadi tidak terima juga, karena orang tua saya mengirim pesan nya lewat grup sekolah tersebut yang banyak orang di dalamnya. Lalu orang tua murid itu menelepon saya karena tidak terima. Cekcoklah orang tua saya dengan orang tua murid melalui telepon, setelah cekcok masalah tersebut diselesaikan secara kepala dingin. Kemudian orang tua si murid tadi merasa janggal pada HP milik anaknya tadi, karena anaknya merasa tidak mengirim pesan apapun ke HP saya. Setelah kami bertukar pikiran, kami memutus kan itu adalah ulah seorang Hecker. Kemudian kami (orang tua dari si murid, orang tua saya, dan saya) berdamai dan saya pun tidak jadi di keluarkan dari sekolah.


Dari kejadiaan diatas saya bertanya ke kakak saya "Bagaimana seorang hecker bisa membajak device (HP) orang lain ?", lalu kakak saya menjawab "Seorang Hecker bisa membajak device dari aktifitas hacking dan menggunakan coding."


Sejak saat itu saya kepo dan ingin belajar tentang coding. Lalu saya mencari di youtube konten yang membahas coding dan menemukan chanel youtube dari Bang Dea Afrizal dan Pak Sandhika Galih. Dari chanel youtube Bang Dea, saya langsung praktek belajar membuat navbar untuk website. Setelah itu saya mendapatkan bootcamp gratis dari Cordova Course di situ saya belajar bahasa Pemogramman Python selama 1 bulan tepatnya bulan Desember 2021 itung-itung mengisi liburan sekolah saya.


Setelah selesai belajar python saya pindah ke chanel youtube Pak Dhika saya belajar coding bahasa Pemogramman JavaScript selama 1 setengah tahun, itu saja saya menyempat-nyempatkan untuk belajar coding sehabis pulang sekolah. Selama 1 setengah tahun itu saya hanya mempelajari HTML, CSS DASAR,CSS 3, CSS LAYOUT, DAN SASS. Setelah itu saya memutuskan untuk vakum 1 - 2 bulan belajar karena boring. Setelah vakum saya kepikiran untuk membuat Portfolio ini yang sedang kalian baca dari melihat youtube. Dari situlah ketertarikan saya pada dunia Teknik informatika bermula, dan berlanjut hingga saat ini. kemudian lanjut ke pertannyaan ke dua.


Pengalaman Apa Saja yang Telah Saya Peroleh?

Selama beberapa tahun terakhir, saya telah mengeksplorasi dunia pemrograman dan teknik informatika, dimulai dari kejadian menarik yang membuat saya penasaran hingga proyek yang telah saya selesaikan. Berikut adalah beberapa pengalaman yang membentuk perjalanan saya:


1. Mengenal Coding Lewat Insiden Tak Terduga

Pandemi COVID-19 membawa perubahan besar dalam kehidupan belajar saya. Pada pertengahan Mei 2020, saya menjadi korban adu domba melalui pesan singkat di chat pribadi yang diduga dikirim oleh seorang hacker. Kejadian ini membuat saya penasaran dan bertanya pada kakak tentang cara kerja coding. Dari sana, saya mulai tertarik mempelajari coding, dan keingintahuan saya di dunia Teknik Informatika tumbuh.


2. Mempelajari Bahasa Pemrograman Pertama: Python dan JavaScript

Ketika pertama kali mulai belajar, saya mengikuti bootcamp online dari Cordova Course dan tutorial di YouTube, seperti dari chanel youtube Bang Dea Afrizal dan Pak Sandhika Galih. Fokus pertama saya adalah Python dan JavaScript, yang membuka pintu untuk memahami dasar-dasar pemrograman python dan javascript. Ini adalah awal penting dalam perjalanan belajar saya.


3. Mengasah Kemampuan Desain Web

Setelah memahami dasar pemrograman, saya ingin mengeksplorasi sisi lain dari dunia IT. Saya mulai belajar HTML, CSS dasar, CSS3, CSS Layout, dan SASS selama 1,5 tahun berikutnya. Sumber utama saya adalah chanel YouTube Pak Sandhika Galih, yang membantu saya memahami konsep-konsep desain web. Dari sinilah saya memulai proyek portofolio sederhana yang menampilkan kemampuan desain web saya.


4. Menantang Diri dengan Proyek Pribadi

Selain belajar dari berbagai chanel, saya juga melakukan berbagai eksperimen dalam proyek pribadi. Salah satunya adalah membangun portofolio online yang merangkum hasil-hasil belajar saya. Meskipun awalnya saya mengalami kendala, ini membantu saya untuk terus mengembangkan keterampilan dan memberikan rasa puas ketika proyek-proyek kecil saya selesai.
